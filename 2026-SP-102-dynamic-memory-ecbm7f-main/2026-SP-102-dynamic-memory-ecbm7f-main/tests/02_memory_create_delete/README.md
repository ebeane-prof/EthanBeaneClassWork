Complete tests for create and delete grid before proceeding to this test

This test uses the `valgrind` tool to check for the presence of memory leaks, invalid read/writes, and other memory-related issues. 

If there is an issue, information is provided to help with your debugging, but in general no tool can tell you the exact source of your problem.

See the [valgrind documentation](https://valgrind.org/docs/manual/mc-manual.html#mc-manual.errormsgs) for more info on how to interpret the feedback.

This [SO post](https://stackoverflow.com/questions/5134891/how-do-i-use-valgrind-to-find-memory-leaks) is about C programming, but it has good advice and links to further reading
